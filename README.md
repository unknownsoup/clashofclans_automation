This project is dedicated to automating the homebase in Clash of Clans, and subsquently maxing your account with nothing but time. I utilize the suggested upgrades in the building and labratory tab to progress and simple army compositons to farm. Also, saved base links to that when the bot upgrades their Town Hall (TH) and has all the new buildings, the base can look pretty again. Ideally, this program will dominate the homebase by keeping builders busy 24/7, and be a top tier donater. 

So far this bot assumes the account is starting at ≥TH11 so electro dragons make the attack sequence easy. Although, you could always get around this by configuring new attack strategies. 

USING THIS BOT:
- This bot utilizes the Andriod Debug Bridge (ADB) library. So, no fruit company named devices.

- I've developed this bot to my device so, you'll have to adjust where this bot is actually tapping depending on the resolution size
of your dedicated device. I'm using a tablet, so the resolution size is bigger, and icons are smaller relative to the screen. The opposite would be true if you connected this to a phone. To make this process easier, you could use the pictures in the icons folder, the findIcon method, and returnCords=True.

LIMITS OF THE BOT:
- I dont see it feasible to try and enable it to attack in war. This bot relies on the attack sequence being dumb simple edrag spam, and war requires calculated troop and spell placement. Same idea when considering the Clan Capital. These clan events require coordination between members that's not impossible, but a little much for a CoC bot.
